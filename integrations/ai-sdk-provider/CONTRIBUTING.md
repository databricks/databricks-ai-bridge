# Contributing

Thank you for your interest in contributing to `@databricks/ai-sdk-provider`.

## Development Setup

```bash
npm install
npm run build
npm test
npm run typecheck
npm run lint
```

## Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run `npm test && npm run typecheck && npm run lint`
5. Commit and push
6. Open a pull request

## Code Style

- Use TypeScript
- Follow existing patterns
- Add tests for new functionality

## Reporting Issues

Open an issue at [GitHub Issues](https://github.com/databricks/databricks-ai-bridge/issues).
